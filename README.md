# Textile Product Management – Frontend

## 📌 Project Overview
This is the frontend UI for the **Textile Product Management System** built using **React.js**.

It allows users to:
- View products
- Add new products
- Place orders
- View order history

The frontend communicates with a **Spring Boot backend** using REST APIs.

---

## 🖥️ Tech Stack
- React.js (Create React App)
- JavaScript (ES6)
- HTML5
- CSS3
- Axios
- Docker
- Vercel (Deployment)
